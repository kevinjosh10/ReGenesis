## 📖 About The Project

For decades, organizations have viewed waste as a liability—something to be hidden, managed, and disposed of at a cost. **ReGenesis** flips this paradigm. 

ReGenesis is an advanced **Circular Economy Engine** that helps businesses bridge the gap between waste generation and product manufacturing. By treating discarded materials as a dormant asset portfolio, our platform empowers organizations to identify highly profitable, sustainable manufacturing opportunities from their exact waste streams.

<br />

## ✨ Key Features

### 1. 🏗️ Command Center (Resource Inventory)
Treat your waste streams as a digital asset portfolio. Log and track available scrap materials (e.g., Lumber, Steel, Textiles). The platform automatically standardizes materials and calculates their baseline scrap value in real-time.

### 2. ⚡ Opportunity Engine
A deterministic intelligence algorithm that analyzes your logged materials against a vast manufacturing knowledge base. Discover exactly which new products you can manufacture based on your current waste inventory.

### 3. 🧪 Resource Composer
A sandbox environment allowing you to experiment with combining multiple waste streams (e.g., Plastic + Metal) to unlock advanced composite products and uncover new cross-industry opportunities.



### 4. 📊 Executive Insights (Resource Intelligence)
Generate McKinsey-style executive reports. Track your **Waste-to-Wealth Index (WWI)**, evaluate 12-month projected cumulative revenue with dynamic charts, and measure direct global impact such as $CO_2$ prevented and water conserved.

- **Animations:** Framer Motion
- **State Management:** Zustand
- **Data Visualization:** Recharts
- **Icons:** Lucide React
- **Routing:** React Router v7

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**
   ```sh
   git clone https://github.com/yourusername/ReGenesis.git
   cd ReGenesis
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Start the development server**
   ```sh
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.
